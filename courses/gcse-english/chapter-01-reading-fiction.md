# Chapter 1: Reading Fiction Texts

## Learning Objectives
- Analyze narrative techniques
- Identify and interpret explicit and implicit information
- Analyze language and structure
- Evaluate effectiveness

## 1.1 Understanding Fiction

**Fiction texts include:**
- Novels and short stories
- Descriptive passages
- Narrative extracts

**Key features:**
- Plot and narrative arc
- Characters and relationships
- Setting and atmosphere
- Themes and messages

## 1.2 Identifying Information (AO1)

**Explicit information:** Directly stated in text
**Implicit information:** Suggested/implied

**Techniques:**
- Quote accurately
- Inference (reading between lines)
- Interpretation
- Selecting relevant detail

**Example question:** "What do we learn about the character's feelings?"
- Find quotes showing feelings
- Explain what they suggest

## 1.3 Language Analysis (AO2)

**Analyze how writers use language:**

**Word choice:**
- Adjectives create description
- Verbs show action
- Adverbs add detail

**Language devices:**
- Metaphor: comparison (e.g., "time is money")
- Simile: comparison using like/as (e.g., "cold as ice")
- Personification: human qualities to non-human
- Alliteration: repeated consonant sounds
- Onomatopoeia: words that sound like their meaning

**Sentence structure:**
- Short sentences: tension, pace
- Long sentences: detail, description
- Questions: engage reader

## 1.4 Structure Analysis (AO2)

**How writers structure texts:**

**Beginning:**
- Hook reader
- Establish setting/character
- Create atmosphere

**Middle:**
- Develop plot
- Build tension
- Character development

**End:**
- Resolution
- Climax
- Reflection

**Structural features:**
- Flashbacks/flash-forwards
- Shifts in time/place
- Perspective changes
- Zooming in/out
- Circular structure

## 1.5 Evaluation (AO4)

**Critically evaluate:**
- Writer's choices
- Effectiveness of techniques
- Impact on reader

**Structure answer:**
- State your opinion
- Support with evidence
- Explain effect
- Consider alternatives

## 1.6 Exam Technique

**Typical questions:**

**Q1:** List four things (4 marks) - 5 minutes
**Q2:** Language analysis (8 marks) - 12 minutes
**Q3:** Structure analysis (8 marks) - 12 minutes
**Q4:** Evaluation (20 marks) - 25 minutes

**Tips:**
- Annotate text
- Use P.E.E. (Point, Evidence, Explain)
- Write in paragraphs
- Use subject terminology
- Support with quotes

## Practice Questions

1. Define explicit and implicit information
2. Name three language devices
3. What effect do short sentences have?
4. How might a writer begin a story to hook the reader?
5. What is P.E.E.?
6. Explain what a metaphor is with an example
7. Why might a writer use a flashback?
8. What should you include in an evaluation?
9. How long should you spend on a 20-mark question?
10. What is the purpose of annotating a text?

## Key Points
✓ Find explicit (stated) and implicit (suggested) information
✓ Analyze language: word choice, devices, sentences
✓ Analyze structure: beginning/middle/end, shifts, perspective
✓ Evaluate: state opinion, support with evidence, explain effect
✓ Use P.E.E. paragraphs
✓ Quote accurately and analyze
