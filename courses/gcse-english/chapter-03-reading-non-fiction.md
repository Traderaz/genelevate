# Chapter 3: Reading Non-Fiction

##Learning Objectives
- Identify purpose and audience
- Analyze presentational features
- Synthesize information
- Compare viewpoints

## 3.1 Types of Non-Fiction

- **Articles:** Newspapers, magazines
- **Letters:** Formal, informal
- **Reviews:** Books, films, products
- **Speeches:** Political, ceremonial
- **Travel writing:** Descriptions, experiences
- **Biographies/autobiographies:** Life stories
- **Leaflets/advertisements:** Persuasive texts

## 3.2 Purpose and Audience

**Purpose (DAFOREST):**
- **D**escribe
- **A**rgue
- **F**orm (inform)
- **O**rganize/order
- **R**eview
- **E**xplain
- **S**tate
- **T**each

**Audience:** Who text is aimed at
- Age group
- Knowledge level
- Interest level
- Formal/informal relationship

## 3.3 Language Analysis

**Persuasive techniques:**
- **Facts:** Proven information
- **Opinions:** Personal views
- **Rhetorical questions:** Don't need answers
- **Emotive language:** Appeals to feelings
- **Statistics:** Numbers for credibility
- **Triplets (rule of 3):** "Education, education, education"
- **Direct address:** "You must act now"
- **Repetition:** Emphasis

## 3.4 Presentational Features

**Headlines:** Catch attention
**Subheadings:** Organize information
**Images:** Visual impact
**Captions:** Explain images
**Bullet points:** List information
**Bold/italics:** Emphasis
**Columns:** Newspaper layout
**Font size/style:** Create hierarchy

## 3.5 Synthesizing Information (AO1)

**Synthesis:** Combining information from multiple sources

**Method:**
1. Read both texts
2. Identify relevant points
3. Select similarities/differences
4. Link ideas across texts
5. Use discourse markers (similarly, however, whereas)

## 3.6 Comparing Writers' Views (AO3)

**Steps:**
1. Identify each writer's viewpoint
2. Find similarities
3. Find differences
4. Support with quotes
5. Analyze methods used

**Comparative phrases:**
- "Similarly..."
- "In contrast..."
- "While Writer A..., Writer B..."
- "Both writers..."
- "However..."

## Key Points
✓ Identify purpose (DAFOREST) and audience
✓ Analyze persuasive techniques (facts, opinions, emotive language)
✓ Evaluate presentational features
✓ Synthesize information from multiple texts
✓ Compare viewpoints and methods
✓ Use comparative language (similarly, in contrast)
